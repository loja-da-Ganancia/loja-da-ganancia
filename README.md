# Projeto-da-disciplina-de-Programa-o-Web-do-CEFET-RJ
Repositório contendo o desenvolvimento de uma aplicação web criada como parte da disciplina de Programação de Software para Web do CEFET/RJ. O projeto é desenvolvido em grupo e contempla etapas de planejamento, modelagem, implementação e versionamento de software, seguindo boas práticas de engenharia de software.
